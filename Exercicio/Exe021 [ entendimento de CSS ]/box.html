Aninhamento dos Box's

Olá Mundo

Segue a anatomia das box's

Todo elemento visível num site, é considerado uma caixa

Toda box tem os elementos básicos:

height (Altura)

width (Largura)

border: é uma linha que segue o contéudo do elemento

padding: é o "preenchimento interno" do elemento, que vai até a border

margin: é o "preenchimento externo" do elemento, que vai fora da border

Tipos de caixa

BOX-LEVEL- sempre pula uma linha, para continuar o continuar ao trocar de caixa
Exemplo de box-level:
div / h1..h6 / p / main / header / nav / article / aside / footer / form / video /

INLINE-LEVEL - Não quebra linha, não ocupa o tamanho inteiro, e segue o padrão na mesma linha
Exemplo de inline-level:
span / a / code / smal / strong / em / sup..sub / label / button / input / select /
